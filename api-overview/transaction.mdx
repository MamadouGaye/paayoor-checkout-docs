L'API Transaction est une API qui vous permet de récupérer la liste des 
transactions(Tout confondu: Collect, Disbursement, Recharge et Withdraw). 

## Endpoints

Vous avez les endpoints suivants qui sont disponibles:

- Recupération de la liste des transactions ([API Référence](https://auth.paayoor.com/register))
