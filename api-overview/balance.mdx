L'API Balance est une API qui vous permet de suivre le solde du compte à tout moment. 
Elle vous donne le solde courant du compte.

## Endpoints

Vous avez les endpoints suivants qui sont disponibles:

- Recupération du solde du compte ([API Référence](https://auth.paayoor.com/register))
