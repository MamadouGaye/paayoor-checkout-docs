L'API Recharge vous permet de recharge le compte Paayoor via les différents cannaux disponibles.

## Flow

L'argent est débité du compte Mobile money / Bancaire pour le positionner dans le compte Paayoor.

## Endpoints

Vous avez les endpoints suivants qui sont disponibles:

- Soumission d'une recharge ([API Référence](https://auth.paayoor.com/register))
- Recupération de la liste des recharges ([API Référence](https://auth.paayoor.com/register))
- Voir les détails d'une recharge ([API Référence](https://auth.paayoor.com/register))

## cannaux disponibles

Les canneaux disponibles pour recharge:

| Countries | Channels         |
| --------- | ---------------- |
| Senegal   | `Wave`           |

## Webhook

Aprés initiation, différents evennements pourront survenir sur une opération(Succeeded, Failed, Canceled). 
A chaque evennement, nous vous enverrons une notification via le callback que vous aurez fourni.

Ci-dessous le format JSON envoyé:

```json
{
    "operation": "recharge",
    "token": "cchow-wxmbw-jrhl5-fxxk2-mcack",
    "status": "succeeded",
    "amount": 1000,
    "pricing": {
        "percentageFee": 1.8,
        "additionalFee": 100,
        "feesAmount": 118,
        "supportedByAccount": true
    },
    "cashInAmount": 1000,
    "transactionAmount": 882,
    "merchantReference": "efenhudshfuidfjhi",
    "initiationDate": "2023-09-06T15:02:44.000000Z"
}
```
