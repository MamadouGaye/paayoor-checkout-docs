L'API Recharge permet de recharger le compte Paayoor via les différents cannaux disponibles.

## Flow

L'argent est débité du compte Mobile money / Bancaire pour le positionner dans le compte Paayoor.

## Endpoints

- Initier une recharge ([API Référence](/api-reference/endpoint/post-recharge))
- Voir les détails d'une recharge ([API Référence](/api-reference/endpoint/get-recharge))
- Recupérer la liste des recharges ([API Référence](/api-reference/endpoint/get-recharges))

## cannaux disponibles

Les canneaux disponibles pour Recharge:

| Countries | Channels         |
| --------- | ---------------- |
| Senegal   | `Wave`           |

## Webhook

Aprés initiation, différents évennements(Succeeded, Failed, Canceled) pourront survenir sur une opération de recharge. 
A chaque évennement, nous enverrons une notification via le callback que vous aurez fourni.
