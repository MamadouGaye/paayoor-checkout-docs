L'API Déboursement permet d'envoyer de l'argent vers les différents cannaux disponibles.

## Flow

L'argent est débité du compte Paayoor pour l'envoyer à un compte Mobile money / Bancaire.

## Endpoints

- Initier un déboursement ([API Référence](/api-reference/endpoint/post-disbursement))
- Voir les détails d'un déboursement ([API Référence](/api-reference/endpoint/get-disbursement))
- Recupérer la liste des déboursements ([API Référence](/api-reference/endpoint/get-disbursements))

## Cannaux disponibles

Les canneaux disponibles pour Déboursement:

| Countries | Channels         |
| --------- | ---------------- |
| Senegal   | `Wave`           |

## Webhook

Aprés initiation, différents évennements(Succeeded, Failed, Canceled) pourront survenir sur une opération de déboursement. 
A chaque évennement, nous enverrons une notification via le callback que vous aurez fourni.
