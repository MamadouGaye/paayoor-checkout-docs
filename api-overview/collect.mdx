---

L'API Collecte permet de recevoir de l'argent via les différents cannaux disponibles.

## Flow

L'argent est débité du compte Mobile money / Bancaire pour le collecter au niveau du compte Paayoor.

## Endpoints

- Initier une collecte ([API Référence](/api-reference/endpoint/post-collect))
- Voir les détails d'une collecte ([API Référence](/api-reference/endpoint/get-collect))
- Recupérer la liste des collectes ([API Référence](/api-reference/endpoint/get-collects))

## cannaux disponibles

Les canneaux disponibles pour Collecte:

| Countries | Channels         |
| --------- | ---------------- |
| Senegal   | `Wave`           |

## Webhook

Aprés initiation, différents évennements(Succeeded, Failed, Canceled) pourront survenir sur une opération de collecte. 
A chaque évennement, nous enverrons une notification via le callback que vous aurez fourni.

Ci-dessous le format JSON envoyé:

```json
{
    "operation": "collect",
    "token": "opxbn-isxnc-dkamk-ujtyc-2ytwg",
    "status": "succeeded",
    "amount": 1000,
    "pricing": {
        "percentageFee": 1.8,
        "additionalFee": 100,
        "feesAmount": 118,
        "supportedByAccount": true
    },
    "cashAmount": 1000,
    "transactionAmount": 882,
    "merchantReference": "kiufjsknduifdnfz",
    "initiationDate": "2023-09-06T15:03:45.000000Z"
}
```
