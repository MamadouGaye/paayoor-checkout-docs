---
title: Retrait
description: "Retrier du compte Paayoor"
---

L'API Retrait permet de retirer de l'argent de son compte Paayoor vers les différents cannaux disponibles.

## Flow

L'argent est débité du compte Paayoor pour l'envoyer à un compte Mobile money / Bancaire.

## Endpoints

- Initier un retrait ([API Référence](/api-reference/endpoint/post-withdraw))
- Voir les détails d'un retrait ([API Référence](/api-reference/endpoint/get-withdraw))
- Recupérer de la liste des retraits ([API Référence](/api-reference/endpoint/get-withdraws))

## Cannaux disponibles

Les canneaux disponibles pour Retrait:

| Countries | Channels         |
| --------- | ---------------- |
| Senegal   | `Wave`           |

## Webhook

Aprés initiation, différents évennements(Succeeded, Failed, Canceled) pourront survenir sur une opération. 
A chaque évennement, nous enverrons une notification via le callback que vous aurez fourni.
