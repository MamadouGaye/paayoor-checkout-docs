---
title: Retrait
description: "Retrier du compte Paayoor"
---

L'API Retrait permet de retirer de l'argent de son compte Paayoor vers les différents cannaux disponibles.

## Flow

L'argent est débité du compte Paayoor pour l'envoyer à un compte Mobile money / Bancaire.

## Endpoints

- Initier un retrait ([API Référence](https://auth.paayoor.com/register))
- Recupérer de la liste des retraits ([API Référence](https://auth.paayoor.com/register))
- Voir les détails d'un retrait ([API Référence](https://auth.paayoor.com/register))

## Cannaux disponibles

Les canneaux disponibles pour Retrait:

| Countries | Channels         |
| --------- | ---------------- |
| Senegal   | `Wave`           |

## Webhook

Aprés initiation, différents évennements(Succeeded, Failed, Canceled) pourront survenir sur une opération. 
A chaque évennement, nous enverrons une notification via le callback que vous aurez fourni.

Ci-dessous le format JSON envoyé:

```json
{
    "operation": "withdraw",
    "token": "okpm2-6fhc3-pcl6q-l0rzd-va7vq",
    "status": "succeeded",
    "amount": 1,
    "pricing": {
        "percentageFee": 1.8,
        "additionalFee": 100,
        "feesAmount": 101,

    },
    "cashAmount": 1,
    "transactionAmount": 102,
    "merchantReference": "djkfiunfiudzjfn",

}
```
