L'API Withdraw vous permet de retirer de l'argent de son compte Paayoor vers les différents cannaux disponibles.

## Flow

L'argent est débité du compte Paayoor pour l'envoyer à un compte Mobile money / Bancaire.

## Endpoints

Vous avez les endpoints suivants qui sont disponibles:

- Soumission d'un retrait ([API Référence](https://auth.paayoor.com/register))
- Recupération de la liste des retraits ([API Référence](https://auth.paayoor.com/register))
- Voir les détails d'un retrait ([API Référence](https://auth.paayoor.com/register))

## Cannaux disponibles

Les canneaux disponibles pour retrait:

| Countries | Channels         |
| --------- | ---------------- |
| Senegal   | `Wave`           |

## Webhook

Aprés initiation, différents evennements pourront survenir sur une opération(Succeeded, Failed, Canceled). 
A chaque evennement, nous vous enverrons une notification via le callback que vous aurez fourni.

Ci-dessous le format JSON envoyé:

```json
{
    "operation": "withdraw",
    "token": "okpm2-6fhc3-pcl6q-l0rzd-va7vq",
    "status": "succeeded",
```
