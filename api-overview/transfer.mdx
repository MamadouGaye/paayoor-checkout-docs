L'API Transfer permet de transférer de l'argent d'un compte d'un canal disponible vers un autre compte d'un cannal disponible.

## Flow

L'argent est transférer du compte Mobile money / Bancaire emmeteur vers un compte Mobile money / Bancaire receveur.

## Endpoints

- Initier un transfert ([API Référence](/api-reference/endpoint/post-transfer))
- Voir les détails d'un transfert ([API Référence](/api-reference/endpoint/get-transfer))
- Recupérer la liste des transferts ([API Référence](/api-reference/endpoint/get-transfers))

## Cannaux disponibles

Les canneaux disponibles pour Tranfert:

| Countries | Channels Emetteurs                  | Channels Receveurs                  | 
| --------- | ----------------------------------- | ----------------------------------- |
| Senegal   |  `Wave`                             |  `Wave`                             |

## Webhook

Aprés initiation, différents évennements(Succeeded, Failed, Canceled) pourront survenir sur une opération de transfert. 
A chaque évennement, nous enverrons une notification via le callback que vous aurez fourni.
