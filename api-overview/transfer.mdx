L'API Transfer vous permet de transférer de l'argent d'un compte d'un canal disponible vers un autre compte d'un cannal disponible.

## Flow

L'argent est débité du compte Mobile money / Bancaire emmeteur vers un compte Mobile money / Bancaire receveur.

## Endpoints

Vous avez les endpoints suivants qui sont disponibles:

- Soumission d'un transfert ([API Référence](https://auth.paayoor.com/register))
- Recupération de la liste des transferts ([API Référence](https://auth.paayoor.com/register))
- Voir les détails d'un transfert ([API Référence](https://auth.paayoor.com/register))

## Cannaux disponibles

Les canneaux disponibles pour tranfert:

| Countries | Channels Emetteurs                  | Channels Receveurs                  | 
| --------- | ----------------------------------- | ----------------------------------- |
| Senegal   |  `Wave`                             |  `Wave`                             |

## Webhook

Aprés initiation, différents evennements pourront survenir sur une opération(Succeeded, Failed, Canceled). 
A chaque evennement, nous vous enverrons une notification via le callback que vous aurez fourni.
