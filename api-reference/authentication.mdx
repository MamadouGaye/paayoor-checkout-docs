---
title: "Authentication"
description: "Endpoint Security"
---

Vous devriez disposer d'une paire de clés d'API que vous allez mettre en headers, comme indiqué ci-dessous, à chaque appel API.

```bash
'API_KEY': 'Api Key'
'SECRET_KEY': 'Api Secret'
```

Ces clés vous donnent accès à tous ces endpoints disponibles sur la documentation.
