---
title: "Authentication"
description: "Sécurité API"
---

Vous devriez disposer d'une paire de clés d'API que vous allez fournir en headers, comme indiqué ci-dessous, à chaque appel API.

```bash
'api-key': 'Api Key'
'secret-key': 'Api Secret'
```

Ces clés donnent accès à tous ces endpoints disponibles sur la documentation.

Toute paire de clé invalide entraine une réponse 401 Unauthorized.
