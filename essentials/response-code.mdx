Quand vous utilisez nos APIs, différents codes de statut(statusCode) pourront être 
retournés. Ci-dessous la liste des code de status avce le message corespondant

| Code   | Statut                | Message                                                                             |
| ------ | --------------------- | ----------------------------------------------------------------------------------- |
| 200    | OK                    | Tout s'est bien passé.                                                              |
| 400    | Bad request           | La requete ne peut etre traitée. Veuillez verifier la requete.                      |
| 401    | Unauthorized          | Vous n'etes pas autorisés à efectuer l'opération. Veuillez vérifier les clés d'API. |
| 403    | Forbidden             | Vous n'avez pas les droits pour effectuer l'opération.                              |
| 404    | Not Found             | La resource n'est trouvée.                                                          |
| 422    | Unprocessable Content | Le contenu de la requete ne correspond pas à ce qui est demandé.                    |
| 500    | Error                 | Quelque chose ne s'est pas bien passé. Contactez nous.                              |
