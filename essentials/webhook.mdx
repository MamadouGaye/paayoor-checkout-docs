Différents évennements(Succeeded, Failed, Canceled) pourront survenir sur une opération. 
A chaque évennement, nous enverrons une notification via le callback que vous aurez fourni.

Le format de données envoyé peut etre différent selon l'operation(Collecte, Déboursement, Transfert, Recharge et Retrait):

- Collecte ([Voir format de données](/api-overview/collect#webhook))
- Déboursement ([Voir format de données](/api-overview/disbursement#webhook))
- Transfert ([Voir format de données](/api-overview/transfer#webhook))
- Recharge ([Voir format de données](/api-overview/recharge#webhook))
- Retrait ([Voir format de données](/api-overview/withdraw#webhook))
