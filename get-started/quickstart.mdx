Suivez ces étapes pour tester nos APIs sur Postman.

### Un compte operationnel

<AccordionGroup>
  <Accordion icon="plus" title="Créer un compte(Si vous n'avez pas de compte)">
    La création de compte Paayoor est trés simple. Rendez-vous [ici](https://checkout.paayoor.com) 
    pour avoir la page de création de compte Paayoor. Remplissez le formulaire. Ensuite appuyer sur le 
    bouton Register. Une fois que le compte est créé avec succès, vous serez invité à confirmer le numéro 
    de téléphone et l'adresse mail saisis. Procéder à leur confirmation pour avoir un compte operationnel.
  </Accordion>
  <Accordion icon="right-to-bracket" title="Se connecter(Si vous avez un compte)">
    Assurez-vous d'avoir les bons accés. Rendez-vous [ici](https://checkout.paayoor.com), renseignez 
    l'adresse mail et le mot de passe puis cliquez sur Se connecter.
  </Accordion>
</AccordionGroup>

### Générer les clés d'API

<AccordionGroup>
  <Accordion icon="key" title="Generate API keys">
    Suivez ce [lien](https://checkout.paayoor.com) pour accéder à la pge de génération de clés. Sur la page, cliquez sur le bouton 
    Générer clés pour avoir le **Api Key** et le **Secret Key** propres à votre compte.
  </Accordion>
</AccordionGroup>

### Télécharger la collection Postman

Cliquez sur le bouton ci-dessous pour télécharger la collection Postman avec laquelle vous pourriez tester les APIs.

<CardGroup cols={3}>
  <Card href="https://checkout.paayoor.com">
      <img src="/images/run-in-postman-button-icon.jpeg" noZoom />
  </Card>
</CardGroup>

Importez la collection et connfigurez un environnement avec les variables suivantes:

- BASE_URL
- API_KEY
- SECRET_KEY

Ça y est, vous pouvez tester à présent.
